1. 我们如何收集信息

 

我们根据法律法规的要求，遵循合法、正当、必要和诚信的原则，仅收集实现产品功能所必要的信息。我们一般会基于以下目的收集和使用您的信息：

 

1.1 为用户注册所需的信息

 

如您仅需非登录状态可使用的基本服务，您有权选择不予注册成为我们的用户并不提供任何信息。若您想体验我们更多的服务，您可以选择注册成为我们的用户，您需要提供您的注册信息（个人邮箱、手机号码等）以完成账号注册并创建密码。

 

1.2 为提供服务所需的信息

 

尊重您的个人隐私是我们的基本原则，我们在此向您详细说明APP对您信息的访问内容，并提供了解释说明。我们会采取各种措施保护您的隐私和个人数据。

 

为使您更好的使用本APP相关服务，我们的以下扩展功能中会收集和使用您的信息。如果您不提供这些个人信息，您依然可以使用，但您可能无法使用部分功能。这些扩展功能包括：

 

蓝牙或 Wi-Fi：

当您使用APP远程管理设备时，需要与通过蓝牙或Wi-Fi连接的设备进行通信。 您可以在使用结束时关闭蓝牙或 Wi-Fi。

 

位置：

当您使用蓝牙功能时，需要开启GPS定位权限才能连接蓝牙进行通讯。使用完毕后，您可以关闭 GPS 定位。

 

相机：

当您输入汽车VIN码并激活VCI时，我们将调用摄像头进行OCR识别，我们承诺不收集和存储图像。

当您上传或更改个人账户头像时，您可以选择使用相机直接拍摄并用于个人形象展示。 为了您的信息安全，我们只会保存您正在使用的图像，并承诺仅将其用于个人图像展示。

 

闪光灯：

当您扫描汽车VIN码时，我们可能会调用设备闪光灯，以便于您在光线不佳的环境下使用扫码功能。

 

震动：

当您扫描汽车VIN码并激活VCI时，我们会调用设备震动权限以提示您已成功激活VCI设备。

 

图像：

当您上传或更改个人账户头像时，您可以选择本地相册中的图片作为您的个人头像。为了您的信息安全，我们只会保存您正在使用的图像，并承诺仅将其用于个人图像展示。

 

视频：

当您向我们发送反馈时，您可以选择将视频发送到我们的应用程序。 为了您的信息安全，我们只会保存您向我们上传的视频，并承诺仅将它们用于改善用户体验。

 

文档：

当您向我们发送反馈时，您可以选择将文件发送到我们的应用程序。为了您的信息安全，我们只会保存您向我们上传的PDF文件，并承诺仅将它们用于改善用户体验。

 

存储空间：

当您配置汽车车型信息时，为方便您随时调用您需要的车型信息，我们会将您操作的车型包存储至云存储空间或本地空间，您可以选择随时删除您缓存的信息。

 

通话状态：

当您在使用过程中有任何问题想联系技术人员获得技术支持时，我们会获取您当前通话设备的IMSI、IMEI、SN信息，以便于确认您的当前通话状态可以联系到我们的技术支持。

 

1.2.3 自启动和关联启动说明


为确保本应用处于关闭或后台运行状态下可正常接收到应用程序推送的信息，本应用需使用（自启动）能力，将存在一定频率或特殊操作（如退出APP时）通过系统发送通知唤醒本应用自启动或关联启动行为，是因为实现功能及服务所必要的。


蓝牙配对：

本应用支持经典蓝牙连接功能，经典蓝牙连接功能需要监听接收系统广播(android.bluetooth.device.action.PAIRING_REQUEST) ，此广播为系统配对广播，如果使用本应用的蓝牙连接功能，本应用须使用自启动能力，通过接收系统广播进行自动配对。

 

2. 我们如何使用您的信息

 

我们严格遵守法律法规的规定及与用户的约定，将收集的信息按照本隐私指引的规定用于以下用途：

我们可能会对产品使用情况进行统计和分析并用于改进我们的产品和服务，并进行必要的业务运营和数据分析，例如运营产品或提供服务，评估、维护和改进产品功能，开发新产品或服务、提供客户支持等，这些统计信息将不会包括您的任何身份识别信息。

 

2.2 安全保障

 

为保障您的账户安全，我们会利用相关信息协助提升APP服务的安全性和可靠性，包括检测、防范和应对可能危害本APP或您的账号的欺诈行为、滥用行为、非法行为、安全风险和技术问题等。

 

2.3 法律原因

 

为了遵守适用法律或回应有效的法律程序，包括来自执法部门或政府机构的程序，以调查或参与民事发现、诉讼或其他对抗性法律程序。

 

2.4 经您许可的其他用途。

 

3. 我们如何存储您的信息

 

3.1 保存期限

 

我们只会在达成本协议所述目的所需的期限内保留您的信息，除非经过您的同意延长保留期或受到法律的允许。

 

如果您注销账号或主动删除上述信息，我们将按照法律、法规规定的最短期限保留您的现有信息，在法律法规要求的最短保存期限内，我们将不会对您的信息进行商业化使用。

 

当您的信息超出上述保存期限，我们将会采取合理步骤以安全方式销毁您的信息。

 

3.2 存放地域

 

您通过使用本APP或其他方式向我们提供信息的行为，视为您同意向鼎匠公司管理的服务器传输和储存包括您的设备信息、数据信息在内的信息。

 

3.3 保护措施

 

我们将采用严格的安全技术措施以及行业通用的程序来确保您的信息不丢失、泄露、毁损或被未经授权的访问、使用，包括：

 

a. 用户敏感信息被加密储存在服务器中，并通过数据隔离技术进行存储；

 

b. 数据传输过程中使用加密传输协议；

 

c. 严格控制数据访问权限，设立完善的敏感数据访问权限申请、审批制度；

 

d. 建立数据安全监控和审计制度，进行全面数据安全控制。

 

3.4 停止运营

 

当本APP发生停止运营的情形时，我们将以推送通知、公告等形式通知您，在合理的期限内删除您的信息，并立即停止收集信息的活动。

 

4. 信息的对外提供与公开

 

我们不会将您的信息提供给任何第三方，但以下情形除外；

 

4.1 获取您的明示同意

 

经您事先单独同意，我们可能会向第三方提供您的信息，我们将向您明确告知对外提供的目的、提供的信息类型，并遵从相关法律法规的要求进行对外提供。

 

4.2 为实现第三方服务所必须

 

为保障本APP的稳定运行、功能实现，使您能够使用和享受更多的服务及功能，我们会根据产品功能实现需要在本APP中嵌入授权合作伙伴的SDK或其他类似的应用程序，我们将审慎的评估第三方的使用目的和使用方式。除另行获得您的单独授权或法律法规另有规定，第三方SDK的运营主体无权将您的信息用于任何其他用途。

6. 您的权利

 

若您选择注册成为我们的用户，您可以按照注册页面提示填写信息且完成全部注册程序后，表示您已成为本APP用户，您可以享受以下权利：

 

6.1 访问您的信息

 

您有权访问您的信息，如果您想行使数据访问权，可以登录本APP进行查看。

 

6.2 更正您的信息

 

当您发现我们处理的关于您的个人信息有错误时，您有权要求我们作出更正。您可以随时提出更正申请。

 

6.3 删除您的信息

 

在以下情形中，您可以向我们提出删除信息的请求：

 

a. 如果我们处理信息的行为违反法律法规；

 

b. 如果我们收集、使用您的信息，却未征得您的同意；

 

c. 如果我们处理信息的行为违反了与您的约定；

 

d. 如果您不再使用我们的产品或服务，或您注销了账号；

 

e. 如果我们不再为您提供产品或服务。

 

6.4 信息主体注销账户

 

您随时可注销此前注册的个人账户，您可以通过本APP的个人信息页面按操作步骤进行注销。在注销账号之后，本APP也将停止为您提供服务，并依据法律法规所规定的最短期限内删除您的所有用户信息。

 

6.5 响应您的上述请求

 

为保障安全，您可能需要提供书面请求，或以其他方式证明您的身份。我们可能会先要求您验证自己的身份，然后再处理您的请求。请您理解，对于某些需要过多技术手段、给他人合法权益带来风险或者非常不切实际的请求，本应用可能会予以拒绝。
